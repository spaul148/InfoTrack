Please enter Search keywords e.g. "efiling integration" and text to search for in the search results e.g. "infotrack". As a first match, the program will try to do a domain match (www.infotrack.com) in the urls of search results. If not found then it will look for a match on the word (infotrack) in the heading. Then it will search the description of search results for a match. The "Found In" column will specify where the search text (infotrack in this case) was found. The "Content" column shows the corresponding text for each match e.g. for a heading match it will show heading text, for a match in description it will show description text and for a url match it will show the heading as well.
Another example to try out would be using Search keywords "cricket" and text to search for "india" within 100 results.
